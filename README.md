## WeChat Web Devtools Beta
WeChat Web Devtools Beta 是微信开发者工具 Beta 版的 Linux 的移植版本，暂时功能仅供测试。

## 微信开发者工具 Beta
微信开发者工具 Beta 是腾讯官方推出的开发者工具 Beta 版本，集成了开发调试、代码编辑及程序发布等功能，让开发者简单的开发和调试小程序和微信网页开发。
